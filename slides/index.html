What's the big deal?

Nothing really. Talk about why it is important, things such as faster queries over file gdb, acid compliance, full database, Cross platform.

Geometry Storage Options

  • ST_Geometry
  • Spatialite
Talk about ST_Geometry for working with Esri products, but if you want true cross-platform support then Spatialite is a great options or if you are working with non-Esri consumers.

Methodology

  • Automated Scripts
  • cProfile to do the timing
  • pstats to process the profile output
Created scripts to automate everything. I did 10 runs on each type of performance factor. I used cProfile to collect the data.
